Seletor Significado Exemplo
Seletor Universal Aplica em todos elementos do documento * { }
seleciona todos os elementos da pagina.
Seletor de Tipo Encontra o elemento pelo nome h1, h2, h3 { }
seleciona os elementos h1, h2, h3
Seletor de Classe Encontra todo elemento que tem atributo
class pelo seu valor especifico. 		.note { }
seleciona todos elementos com valor
do atributo class igual a 'note'
p.note { }
seleciona todos elementos paragrafo com valor
do atributo class igual a 'note'
Seletor de ID Encontra todo elemento que tem atributo
id pelo seu valor especifico. 		#introducao { }
seleciona o elemento que tem o valor do
atributo id igual a introducao
