Attribute Selectors

Seletor Significado Exemplo
Existência


[ ]
Corresponde a um atributo específico
(qualquer que seja o seu valor) 		p[class]
Segmenta qualquer elemento <p> com um
atributo chamado classe
Igualdade


[=]
Corresponde a um atributo específico com
um valor específico 		p[class="intro"]
Segmenta qualquer elemento <p> com
atributo de classe chamado quais o valor seja intro
Espaço



[~=]
Corresponde a um atributo específico
cujo valor aparece em um espaço-
lista separada de palavras 		p[class~="intro"]
Segmenta qualquer elemento <p> com
um atributo chamado classe cujo
valor é uma lista de espaços separados
palavras, uma das quais é intro
Prefixo



[^=]
Corresponde a um atributo específico
cujo valor começa com um
string específica 		p[attr^"d"]
Segmenta qualquer elemento <p> com
um atributo cujo valor começa
com a letra "d"
Substring



[*=]
Corresponde a um atributo específico
cujo valor contém um específico
substring 		p[attr*"do"]
Segmenta qualquer elemento <p> com um
atributo cujo valor contém
as letras "do"
Sufixo



[$=]
Corresponde a um atributo específico
cujo valor termina com um específico
de string 		p[attr$"g"]
Segmenta qualquer elemento <p> com um
atributo cujo valor termina com
a letra "g"
