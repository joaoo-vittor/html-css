Seletor Significado Exemplo
Seletor Universal Aplica em todos elementos do documento * { }
seleciona todos os elementos da pagina.
Seletor de Tipo Encontra o elemento pelo nome h1, h2, h3 { }
seleciona os elementos h1, h2, h3
Seletor de Classe Encontra todo elemento que tem atributo
class pelo seu valor especifico. 		.note { }
seleciona todos elementos com valor
do atributo class igual a 'note'
p.note { }
seleciona todos elementos paragrafo com valor
do atributo class igual a 'note'
Seletor de ID Encontra todo elemento que tem atributo
id pelo seu valor especifico. 		#introducao { }
seleciona o elemento que tem o valor do
atributo id igual a introducao
Seletor filho Encontra todo elemento que é filho
direto do outro. 		li > a { }
seleciona o elemento 'a' que tem são filhos
diretos de 'li'
Seletor de descendente Corresponde a um elemento que é um
descendente de outro especificado
elemento (não apenas um filho direto de
aquele elemento) 		p a { }
Tem como alvo quaisquer elementos 'a' que
esteja dentro de um elemento 'p', mesmo que
existem outros elementos aninhados
entre eles
Seletor de irmão adjacente Corresponde a um elemento que é o
próximo irmão de outro 		h1 + p { }
Segmenta o primeiro elemento 'p'
após qualquer elemento 'h1' (mas não
outros elementos 'p')
Seletor de irmão geral Corresponde a um elemento que é um
irmão de outro, embora seja
não tem que ser diretamente
elemento precedente 		h1 + p { }
Se você tivesse dois elementos 'p' que
são irmãos de um elemento 'h1',
esta regra se aplicaria a ambos
